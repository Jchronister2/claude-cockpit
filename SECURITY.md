# Security

Do not include session transcripts, browser profiles, cookies, tokens, screenshots, or local project paths in bug reports.

The optional cookie bridge is disabled unless `COCKPIT_COOKIE_BRIDGE_TOKEN` is set to a value of at least 32 characters. Treat that value and all imported cookies as credentials.

For a private vulnerability report, contact the repository owner through the email address on the GitHub profile instead of opening a public issue.
